## Technical Implementation

### Changed Properties (EateryCard.tsx)

```tsx
// Restaurant Name - NEW
fontFamily: "'Georgia', 'Garamond', serif"
fontWeight: 300
letterSpacing: '0.02em'
// From: standard serif, inherit weight, tight tracking
// To: elegant serif, light weight, generous spacing

// Rating/Category - NEW
fontFamily: "-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif"
fontWeight: 300
letterSpacing: '0.01em'
// From: generic font, default weight, no spacing
// To: system font, light weight, subtle spacing

// Location - NEW
fontFamily: "-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif"
fontWeight: 300 (implied)
letterSpacing: '0.005em'
// From: generic font, default weight, no spacing
// To: system font, light weight, minimal spacing

// Buttons - NEW
fontFamily: "-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif"
fontWeight: 500
letterSpacing: '0.02em'
// From: generic font, semibold, no spacing
// To: system font, medium, professional spacing

// Badges - NEW
fontFamily: "-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif"
fontWeight: 500-600
letterSpacing: '0.12em'
// From: generic font, bold, minimal spacing
// To: system font, medium-bold, compressed spacing
```
