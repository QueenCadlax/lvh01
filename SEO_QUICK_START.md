## 📞 TROUBLESHOOTING

**Issue:** Sitemap not recognized by Google
**Solution:** Sign up for Google Search Console → Submit sitemap manually

**Issue:** Image not showing on Facebook share
**Solution:** Make sure `/public/og-image.png` exists (1200x630px)

**Issue:** HTTPS redirect not working
**Solution:** Your hosting provider needs to support .htaccess OR use nginx config instead

**Issue:** PWA not installable
**Solution:** Make sure manifest.json is valid + service worker (sw.js) exists
